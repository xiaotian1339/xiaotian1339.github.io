第二题

（简单）编程实现某中药房药材按使用频率存放。使用频率越高的药，存放到越近的区域。其中A区存放频率大于或等于50%的药，B区存放使用频率为40%-50%的药，C区存放使用频率为30%-40%的药，D区存放使用频率为20%-30%的药，E区存放使用频率小于20%的药。 
#include<stdio.h>
main()
{
	float pl;
	int a,b,c,d;
	a=50,b=40,c=30,d=20;
	printf("请输入该药材使用频率(%%)：\n");
	scanf("%f",&pl);
	if (pl>=a)//放入A区
	{
		printf("该药品使用频率很高，请放入A区\n");
	}
	else if (pl>=b)//放入B区
	{
		printf("该药品使用频率较高，请放入B区。\n");
	}
	else if (pl>=c)//放入C区
	{
		printf("该药品使用频率中等，请放入C区\n");
	}
	else if(pl >=d)//放入D区
	{
		printf("该药品使用频率较低，请放入D区\n");
	}
	else//放入E区
	{
		printf("该药品使用频率过低，请放入E区。\n");
	}
