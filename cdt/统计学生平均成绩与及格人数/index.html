**统计学生平均成绩与及格人数**

*统计学生平均成绩与及格人数

本题要求编写程序，计算学生们的平均成绩，并统计及格（成绩不低于60分）的人数。题目保证输入与输出均在整型范围内。

输入格式:

输入在第一行中给出非负整数N，即学生人数。第二行给出N个非负整数，即这N位学生的成绩，其间以空格分隔。

输出格式:

按照以下格式输出：

caverage = 成绩均值
count = 及格人数

其中平均值精确到小数点后一位。

输入样例: 

5
77 54 92 73 60

输出样例: 

average = 71.2
count = 4

解题代码： 


#include<stdio.h>
int main()
{
  int n;
  scanf("%d",&n);
  if(n>=0){
  	//判断输入是否为0人
    if(n==0){
    printf("average = 0.0\n");
    printf("count = 0");

	}
    else{
 	int i;
	int sum=0;
	int temp=0;
	for(i=1;i<=n;i++)
  {
    int t;
    scanf("%d",&t);
    sum+=t;
    if(t>=60) temp++;
  }
  printf("average = %.1lf\n",1.0*sum/n);
  printf("count = %d\n",temp);}}
  return 0;
}
